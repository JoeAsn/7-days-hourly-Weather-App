# 🌤️ Weather App

A responsive weather web app built with HTML, CSS, and JavaScript that detects 
your current location automatically and displays real-time weather data.

## Features
- 📍 Auto-detects current location using Geolocation API
- 🌍 Search weather for any city worldwide
- 📅 7-day weather forecast
- ⏱️ Hourly temperature breakdown
- 🌡️ Metric and Imperial unit toggle
- 🔍 City autocomplete suggestions

## APIs Used
- [OpenWeatherMap](https://openweathermap.org/) — current weather data
- [Open-Meteo](https://open-meteo.com/) — 7-day & hourly forecast
- [Rest Countries](https://restcountries.com/) — country autocomplete

## Tech Stack
- HTML / CSS / JavaScript
- Fetch API
- Browser Geolocation API
